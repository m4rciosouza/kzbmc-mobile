Sincronizar Dados

Sincronizando o seu Canvas de Modelo de Negócios em nosso sistema online você será capaz de editar os seus modelos canvas online e que qualquer lugar ou dispositivo, compartilhar com seus amigos, imprimi-lo, além de manter uma cópia de segurança de seu Canvas.

Para sincronizar os dados cadastre-se gratuitamente aqui para obter sua API Key, um código necessário para identificar e sincronizar seus modelos canvas em sua conta online.

* Cadastre-se gratuitamente aqui, digite sua API Key acima e clique em "Sincronizar Dados" para expandir as possibilidades de seu modelo canvas.

Instalar Aplicativo

Para instalar o aplicativo Canvas de Modelo de Negócios no seu iPhone/iPad clique no botão de bookmarking, que dependendo da versão do seu IOS será ou ( clique em Add to Home Screen ) , um atalho será criado na sua área de trabalho para o acesso ao aplicativo.
